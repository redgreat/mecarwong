半径 r，轮间距 L，轴距 W

w1 = (1/r)*( vx - vy - (L+W)*ω )   # 右前
w2 = (1/r)*( vx + vy + (L+W)*ω )   # 左前
w3 = (1/r)*( vx + vy - (L+W)*ω )   # 左后
w4 = (1/r)*( vx - vy + (L+W)*ω )   # 右后

其中 vx [-1,1]：前/后，vy [-1,1]：左/右，ω [-1,1]：自转。


硬件连线与电源

电机驱动板 (DFR DRI0039) 插在 Spresense 扩展板 上（非主板直接插）。
DRI0039 的 IO 电平 5 V，而 Spresense 为 3.3 V：
• 板子已带 74HC595 + IRF 驱动，逻辑端能识别 3.3 V，高可靠起见可在 Spresense 与盾之间串 1 kΩ 电阻。
12 V/5 A 供电接到 DRI0039 VIN，Spresense 用 USB 供电即可（或主板上的 5 V PIN 由同一电池降压供电）。
